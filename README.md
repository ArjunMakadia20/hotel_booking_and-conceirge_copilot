# Hotel Booking and Concierge Copilot

AI/ML internship project focused on hotel booking analytics and concierge copilot development.

## Project Overview

This project uses hotel booking and hotel review datasets to:

- Analyze booking patterns and customer behavior
- Perform data cleaning, preprocessing, and exploratory data analysis (EDA)
- Build and evaluate machine learning models for hotel booking predictions
- Identify key factors influencing booking outcomes
- Develop explainable AI insights using SHAP
- Support future concierge copilot and recommendation system development

## Datasets

1. Hotel Booking Demand Dataset
   - hotel_bookings.csv

2. TripAdvisor Hotel Reviews Dataset
   - tripadvisor_hotel_reviews.csv
